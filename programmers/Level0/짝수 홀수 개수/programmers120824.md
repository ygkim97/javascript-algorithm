## 코딩테스트 연습 > 코딩테스트 입문 > 짝수 홀수 개수

https://school.programmers.co.kr/learn/courses/30/lessons/120824

---

### 문제 설명

정수가 담긴 리스트 num_list가 주어질 때, num_list의 원소 중 짝수와 홀수의 개수를 담은 배열을 return 하도록 solution 함수를 완성해보세요.

</br>

### 제한사항

- 1 ≤ num_list의 길이 ≤ 100
- 0 ≤ num_list의 원소 ≤ 1,000

</br>

### 입출력 예

| numbers     | result |
| ----------- | ------ |
| [1,2,3,4,5] | [2,3]  |
| [1,3,5,7]   | [0,4]  |
