## 코딩테스트 연습 > 코딩테스트 입문 > 두 수의 차

https://school.programmers.co.kr/learn/courses/30/lessons/120803

---

### 문제 설명

정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

</br>

### 제한사항

- -50000 ≤ num1 ≤ 50,000
- -50000 ≤ num2 ≤ 50,000

</br>

### 입출력 예

| num1 | num2 | result |
| ---- | ---- | ------ |
| 2    | 3    | -1     |
| 100  | 2    | 98     |
